❌ 1. 첫 시도 실패 — 책상 사진 3장

서로 너무 다른 방향에서 찍음

공통 특징점이 거의 없음

스티칭 결과가 심하게 왜곡되거나 완전 실패

오류 메시지:

Stitch failed! status code: 1

🔍 원인

평면(책상·노트북·컵)은 반복 텍스처와 동일 패턴이 많아 특징점을 잘 찾지 못함

카메라 위치 변화가 크고 시야가 맞지 않음

❌ 2. OpenCV Stitcher 실패

단순 Stitcher 코드도 status 1 반환 → 실패
원인은 동일 : overlap 부족 + feature 부족

✔️ 3. 해결 — 특징점 많은 실외 이미지로 재촬영

다음 조건을 모두 만족시켜 성공

좌 → 중 → 우 순서

동일한 위치에서 카메라만 수평 이동

간판, 건물 라인 등 특징점 많음

겹치는 영역 충분

결과: 완전한 파노라마 생성 성공
